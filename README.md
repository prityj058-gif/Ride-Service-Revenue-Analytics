# 🚖 Ride Service Revenue Analytics

## 📌 Project Overview

This project analyzes ride service data to understand revenue performance, ride trends, and payment behavior using Power BI.
The goal is to transform raw ride data into an interactive dashboard that provides clear business insights for better decision-making.

---

## 🎯 Objective

The main objectives of this project are:

* Analyze total revenue generated from rides
* Understand ride patterns based on date and time
* Identify the most used payment methods
* Analyze ride distance and duration trends
* Create an interactive Power BI dashboard for business insights

---

## 🛠 Tools Used

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation

---

## 📊 Data Features

The dataset contains the following features:

* `ride_id` – Unique ID for each ride
* `date` – Ride date
* `time` – Ride time
* `ride_status` – Status of ride (completed, cancelled, etc.)
* `source` – Ride starting location
* `destination` – Ride ending location
* `distance` – Distance of ride
* `duration` – Duration of ride
* `ride_charge` – Base ride charge
* `misc_charge` – Additional charges
* `total_fare` – Total fare of the ride
* `payment_method` – Payment method used

---

## 🧹 Data Cleaning Steps

Data preparation was performed using **Power Query**:

* Removed duplicate records
* Checked and handled missing values
* Corrected data types (date, time, numeric values)
* Created structured dataset for dashboard analysis

---

## 📈 Dashboard Insights

The Power BI dashboard includes the following visual insights:

* Total Revenue Generated
* Total Number of Rides
* Details Of Payment Method
* Ride Status 
* Distance vs Duration Analysis
* Total Fare by Distance and Ride Time

---

## 🔍 Key Insights

* Identified the most frequently used payment methods
* Observed revenue contribution from ride and miscellaneous charges
* Analyzed ride distribution based on distance and duration
* Found trends in ride demand across different dates and times
